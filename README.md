# Cumulonimbus

A slapdash node.js uploading adventure

## Installation

Prerequisites:

* node.js (0.6.x)
* npm - get it at http://npmjs.org

Run:

    npm install
    ./build.sh
    open http://localhost:1337

Kill:
   cat .pid | xargs kill

Notes:

* I have not tested this on IE
* Despite my hopes and dreams, this may not be web-scale!
